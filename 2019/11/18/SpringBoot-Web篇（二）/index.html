总的来说，内容如下：

@Aspect: 标明该类为切面类
@Pointcut: 设置切入点
@Before: 进入方法前调用
@After: 方法结束后调用
@AfterReturning: 有返回值才调用，且在@After前调用，并获取到result（返回值）
@AfterThrowing: 抛异常时调用，不会调用@AfterReturning和不会调用around（因为没有返回值）
@Around: 有返回值才调用，对返回数据进行处理，在@AfterReturning前调用
